1. 平台简介

236|

GyneCold AI 是一个面向转录组差异表达分析的智能研究平台。平台由三部分组成：

237|
    238|
  • GyneCold AI 前端网站 — 你当前看到的界面，部署在 GitHub Pages 上，提供注册登录和对话入口
    • 239|
  • AI Agent 后端 — 大语言模型（DeepSeek Chat）驱动的分析助手，理解自然语言并自动调用分析工具
    • 240|
  • autoSX 分析引擎 — 后端的生物信息学分析管线，实现 limma 差异表达检验、火山图/热图/PCA 可视化
    • 241|
242|

你只需要用中文告诉 AI 你想做什么，Agent 会自动完成文件上传、参数配置、任务提交、结果获取等全部流程。

5. 配置分析参数

377|

RNA-seq 差异表达分析支持以下可调参数，你可以通过自然语言告诉 AI 来设置：

378| 379|

5.1 参数一览

380| 381| 382| 383| 384| 385| 386| 387| 388| 389| 390| 391| 392| 393| 394| 395| 396| 397| 398| 399| 400| 401| 402| 403| 404| 405| 406| 407| 408| 409| 410| 411| 412| 413| 414| 415| 416| 417| 418| 419| 420|
参数默认值说明
p_value_threshold0.05显著性水平阈值，越小越严格
log2fc_threshold1.0差异倍数（log2）阈值，绝对值需大于此值
test_methodlimma统计方法：limma / ttest / mannwhitney
normalizationcpm标准化：none / cpm / tpm / quantile
log_transformauto是否 log2 转换：auto / yes / no
min_samples_expr0低表达基因过滤（N 个样本中表达量>0）

6.2 查看图表

454|

火山图：展示基因的差异表达分布。横轴为 log2FC（差异倍数），纵轴为 -log10(padj)（显著性）。左上和右上区域为显著差异基因。

455|
    456|
  • 红色/绿色点 = 显著上调/下调的基因（padj < 阈值）
    • 457|
  • 灰色点 = 不显著基因
    • 458|
  • 被标注的基因是变化最显著的候选基因
    • 459|
460| 461|

热图：展示显著差异基因在样本间的表达模式。行 = 基因，列 = 样本，颜色表示 Z-score 标准化后的相对表达水平。

462|
    463|
  • 红色 = 高表达，蓝色 = 低表达
    • 464|
  • 顶部样本标签颜色区分不同实验组
    • 465|
  • 横向聚类反映基因表达模式的相似性
    • 466|
467| 468|

PCA 图：主成分分析降维可视化，展示样本间的全局相似性。

469|
    470|
  • 每个点代表一个样本，颜色代表分组
    • 471|
  • 椭圆表示 95% 置信区间
    • 472|
  • 横纵轴标注方差解释率（PC1/PC2）
    • 473|
  • 两组样本在 PCA 空间分离越好，表示差异越明显
    • 474|
