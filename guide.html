1. Platform Overview

GyneCold AI is an AI-powered multi-omics integrative analysis platform designed for Traditional Chinese Medicine (TCM) syndrome research. It combines Large Language Models (DeepSeek) with bioinformatics analysis pipelines to enable natural language-driven omics data analysis.

Key capabilities:

  • RNA-seq Differential Expression — limma-based analysis with publication-ready volcano/heatmap/PCA plots
  • Mendelian Randomization — TwoSampleMR with bidirectional analysis, MR-PRESSO, and HTML reports
  • Multi-Omics Integration — GWAS, eQTL, DNA methylation, microbiome, proteomics, metabolomics
  • Spatial Transcriptomics — 空间转录组整合分析，组织切片基因表达空间映射与区域差异解析
  • Single-cell Analysis — Scrublet 双细胞检测、Harmony 去批次、UMAP/t-SNE 聚类、差异基因与通路富集、DEG 间 MR 分析，支持 10X / Smart-seq2 / Drop-seq
  • Virtual Cell — 数字细胞模拟平台，基于 LLM + 生信管线预测扰动前后的转录组变化、基因调控网络扰动模拟、先验知识驱动的虚拟 CRISPR / 药物干预实验设计
  • Cell-cell Communication — 基于 ligand-receptor 数据库（CellChatDB / CellPhoneDB / NicheNet）的细胞间通讯网络推断与可视化，支持空间邻近约束通讯和疾病特异性通讯重编程分析
  • TWAS / PWAS / Co-localization — 组织特异性转录组关联分析、蛋白质组关联分析、共定位分析（COLOC / eCAVIAR）
  • Multi-omics Factor Analysis — MOFA+ 多模态因子分解、WGCNA 共表达网络、共定位网络构建
  • Causal Inference with Genetics — Mediation MR、双向 MR、多变量 MR、Network MR、Drug-target MR
  • Knowledge Graph Construction — 基于文献挖掘与多组学证据链的疾病-基因-通路-表型知识图谱自动化构建

3. Using the AI Assistant

After signing in, you'll enter the full-screen chat interface. Simply describe your analysis needs in natural language:

"Run differential expression analysis on my RNA-seq data"
"Perform Mendelian Randomization with exposure file and outcome file"
"Use limma method, p-value threshold 0.01"
"List available analysis modules"

The AI Assistant will automatically interpret your request and execute the corresponding analysis pipeline.

5. Mendelian Randomization

Leverage GWAS summary statistics to infer causal relationships between exposures and outcomes.

Supported Methods

  • IVW (Inverse Variance Weighted)
  • MR Egger
  • Weighted Median
  • Weighted Mode
  • Simple Mode
  • MR-PRESSO (Global test & outlier correction)

Features

  • Bidirectional MR (Trait A ↔ Trait B)
  • Local PLINK LD clumping
  • Automatic column detection & schema repair
  • Comprehensive HTML reports with Chinese interpretation

7. FAQ

Q: Why can't I connect to the AI service?

The AI Agent backend needs to be running separately. Contact your system administrator.

Q: What file formats are supported?

CSV, TSV, TXT for expression data. VCF, CSV, TSV for GWAS summary statistics.

Q: How long does an analysis take?

RNA-seq DE: 1-3 minutes. MR: 3-10 minutes depending on data size.
